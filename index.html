Caso de Éxito

Optimización de Alto Impacto: Sistema de Gestión Veterinaria

Contexto: Durante mis prácticas (FCT), asumí el mantenimiento y mejora de un sistema crítico desarrollado en PHP y Vue 3.

El Reto: El sistema presentaba una degradación severa en las búsquedas, con latencias de hasta 8 segundos. Identifiqué que el origen era un problema de consultas N+1 y una estructura de base de datos no optimizada para el volumen de datos actual.

Ebersy — Sistema integral de gestión para centros veterinarios.

Acción:

Ejecuté una refactorización profunda del backend, eliminando las consultas redundantes y rediseñando la persistencia mediante índices estratégicos y triggers en PostgreSQL. Además, optimicé el estado de la aplicación en Vue 3 para reducir la carga innecesaria en el cliente.

Resultado: Logré una reducción drástica del tiempo de respuesta, pasando de 8 segundos a 0.23ms (mejora >3000%).
