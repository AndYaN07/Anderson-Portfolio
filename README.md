# Portfolio Personal - Anderson Guanche

Este repositorio aloja el código fuente de mi sitio web personal. El proyecto nace como entrega final para el módulo de Desarrollo Web del **Master Prometeo**, sirviendo como base fundacional para mi presencia digital profesional.

## 🚀 Sobre el proyecto

El objetivo es presentar un sitio web estático, limpio y funcional que cumpla con los requisitos académicos actuales, pero diseñado con una estructura que permita su escalabilidad futura.

**Estado actual:** Versión 1.0 (MVP Académico).

## 🛠️ Tecnologías utilizadas

* **HTML5:** Estructura semántica y accesible.
* **CSS3:** Estilizado personalizado y diseño responsive.

## 🔮 Roadmap y Futuras mejoras
