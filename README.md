# Portfolio Personal - Anderson Guanche

Este repositorio alberga la evolución de mi sitio web personal. Originalmente concebido como el proyecto final para el **Master Prometeo**, ha evolucionado de un sitio estático a una plataforma dinámica que muestra mis capacidades técnicas y proyectos recientes.

## 🚀 Sobre el proyecto

El objetivo es ofrecer una experiencia de usuario fluida y moderna. Con la última actualización, el sitio ha dejado de ser meramente informativo para integrar lógica de programación que gestiona la interactividad y la visualización dinámica de contenidos.

**Estado actual:** Versión 2.0 (Funcionalidades dinámicas implementadas).

## 🛠️ Tecnologías utilizadas

- **HTML5:** Estructura semántica avanzada enfocada en SEO y accesibilidad.
- **CSS3:** Diseño responsivo (Mobile First), animaciones y estilos personalizados.
- **JavaScript (ES6+):** \* `app.js`: Gestión de interactividad general y lógica de la interfaz.
  - `projects.js`: Gestión dinámica de la carga y renderizado de proyectos.

## 📈 Mejoras implementadas (v2.0)

Tras la última fase de desarrollo, se han completado los siguientes hitos del Roadmap:

- [x] **Interactividad con JS:** Implementación de lógica personalizada para mejorar la UX.
- [x] **Gestión dinámica de proyectos:** Los proyectos ahora se gestionan mediante scripts independientes (JSON/objetos) para facilitar el mantenimiento.
- [x] **Refactorización de Estilos:** Limpieza y expansión de `style.css` para un diseño más robusto.
- [x] **Despliegue optimizado:** Configuración y migración para visualización en Vercel.

## 🔮 Roadmap futuro

- [ ] Implementación de Modo Oscuro (Dark Mode).
- [ ] Optimización avanzada de rendimiento (Core Web Vitals).
